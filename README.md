# ZPTimer
本Demo主要介绍了NSTimer和CADisplayLink这来两种定时器的用法，还介绍了在创建定时器的过程中造成的定时器和视图控制器相互引用的问题是如何产生的以及是如何解决的。还介绍了iOS中NSProxy类的作用以及用法。
